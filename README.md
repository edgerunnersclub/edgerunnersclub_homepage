# EdgeRunners Club - 회사 홈페이지

AI 에이전트 개발, 소프트웨어 개발, AI 활용 교육을 제공하는 SMB(Small and Midsize Business) 전문 기술 회사의 공식 홈페이지입니다.

## 🚀 특징

- **모던한 디자인**: 블랙앤화이트 컨셉의 미니멀하고 전문적인 디자인
- **반응형 레이아웃**: 모바일, 태블릿, 데스크톱 모든 기기에서 완벽하게 작동
- **부드러운 애니메이션**: 스크롤 기반 애니메이션과 인터랙션
- **최적화된 성능**: 순수 HTML, CSS, JavaScript로 구현된 빠른 로딩 속도
- **로고 통합**: EdgeRunners Club 로고가 네비게이션과 푸터에 적용

## 📋 섹션 구성

1. **Hero Section**: 메인 비주얼 및 CTA
2. **Services**: 세 가지 핵심 서비스 소개
   - AI Agent 개발
   - Software Development
   - AI 활용 교육
3. **References**: 완료된 프로젝트 레퍼런스 (6개 프로젝트 사례)
   - 스마트 카페 관리 시스템
   - 온라인 쇼핑몰 통합 플랫폼
   - AI 고객 상담 챗봇
   - 음식점 POS & 배달 통합
   - AI 활용 실전 워크샵
   - 스마트 재고 관리 시스템
4. **About**: 회사 소개 및 주요 통계
5. **Contact**: 문의 폼 및 연락처 정보

## 🚀 사용 방법

1. 이 저장소를 클론하거나 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 로컬 서버를 사용하려면:
   ```bash
   # Python을 사용하는 경우
   python -m http.server 8000
   
   # Node.js를 사용하는 경우
   npx serve
   ```

## 🎨 커스터마이징

### 색상 변경

`style.css` 파일의 `:root` 섹션에서 CSS 변수를 수정하여 색상을 변경할 수 있습니다:

```css
:root {
    --primary: #000000;
    --secondary: #666666;
    --bg-dark: #ffffff;
    --bg-darker: #f5f5f5;
    /* ... */
}
```

현재는 블랙앤화이트 미니멀 컨셉으로 디자인되어 있습니다.

### 콘텐츠 수정

`index.html` 파일에서 텍스트, 이미지, 링크를 직접 수정할 수 있습니다.
